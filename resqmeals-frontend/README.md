# ResQMeals Frontend

The client-side application for the ResQMeals platform, built with React, TypeScript, and Vite.

## Setup Instructions

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run the Application**:
   ```bash
   npm run dev
   ```

3. **View in Browser**:
   Open `http://localhost:5173`

## Configuration

The frontend connects to the backend at `http://localhost:8000`. You can update the API base URL in `src/lib/api.ts`.

## Tech Stack

- **React 19**: Modern UI library
- **Tailwind CSS**: For styling and responsive design
- **Framer Motion**: For smooth animations
- **Lucide React**: For icons
- **React Router**: For navigation
